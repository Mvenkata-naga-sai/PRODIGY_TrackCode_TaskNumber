# 🔐 User Authentication System

## 📌 Overview
This project is a **Node.js Authentication API** that supports user registration, login, and protected routes using **JWT (JSON Web Token)**. It also includes **role-based access control (RBAC)** for admin functionality.

---

## 🚀 Features
- User Registration
- User Login
- Password Hashing (bcrypt)
- JWT Authentication
- Protected Routes
- Role-Based Access (Admin/User)

---

## 🧰 Tech Stack
- Node.js
- Express.js
- MongoDB (Mongoose)
- bcryptjs
- jsonwebtoken
- dotenv
